# Node 22.22.1 Upgrade Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Upgrade all Node.js references from 20.20.1 to 22.22.1, validating locally before updating CI and production.

**Architecture:** Two-phase approach. Phase 1 updates local dev tooling (.nvmrc, Dockerfiles, docker-compose, package.json engines, @types/node) and validates with a local build + Docker Compose run. Phase 2 updates GitHub Actions workflows and all documentation references, then relies on existing CI + ArgoCD to deploy.

**Tech Stack:** Node.js 22.22.1, nvm, GitHub Actions, Docker, pnpm, npm
